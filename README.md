# The Wild Update
this modpack is the wild update for mineclone 2 and 5, features all the new thing which minecraft 1.19 has in it.

this will have -
- Mangrove full set + roots
- Mangrove swamps
- All sculk Blocks
- Deep Dark
- Ancient cities
- Chest Boats
- Frogs
- Tadpoles
- Fireflies
- Warden
and more
